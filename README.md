# GPU_SingleCell_Setup
Guides to create Conda environments for single cell analysis that leverage GPU acceleration for faster computing.

Tools included:
* Rapids-Suite
* scanpy
* muon
* scvi-tools
* pytorch
* jax
* rapids-singlecell
* pyMDE
* leiden (cpu)
* louvain (cpu)
* force atlas 2 (cpu)
* Decoupler (cpu)
* squidpy (cpu)
